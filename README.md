# React + Vite

This dashboard analyzes a FIFA Player Performance & Market Value dataset. 

https://www.kaggle.com/datasets/itszubi/fifa-player-performance-and-market-value

To run this application locally, simply clone this repo and run 

```
npm install
npm start
npm run dev
```

Website can be found here

https://lshen123.github.io/FIFA_Sports_Player_Dashboard/
